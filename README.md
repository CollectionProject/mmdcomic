# MMDComic 

MMDComic is a online tool for Making comic via mmd models just in browser. 

## Install

```
git clone git@github.com:easychen/mmdcomic.git
cd mmdcomic
php -S localhost:8888 route.php
```

Open chrome visiting http://localhost:8888/editor
